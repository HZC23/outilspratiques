    Comment utiliser le dictionnaire

    1. Tapez un mot dans la barre de recherche
    2. Sélectionnez la langue souhaitée
    3. Cliquez sur le bouton de recherche ou appuyez sur Entrée
    4. Le résultat affichera les définitions, synonymes et exemples

    Fonctionnalités

    • Recherche de définitions via l'API DicoLink (français)
    • Recherche de définitions en anglais via Free Dictionary API
    • Dictionnaire local de secours en cas d'échec des API
    • Historique des recherches récentes

    Sources de données

    • DicoLink API : API principale utilisée pour le français
    • Free Dictionary API : API secondaire pour les définitions en anglais
    • Dictionnaire local : Utilisé uniquement en cas d'échec des API externes

    Langues supportées

    • Français (fr)
    • Anglais (en)
    • Espagnol (es)
    • Allemand (de)
    • Latin (la)
